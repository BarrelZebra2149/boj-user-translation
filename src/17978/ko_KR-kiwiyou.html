세탁기

문제

세탁기에 n 벌의 옷이 있다. 세탁기는 모든 옷을 한 번에 담을 만큼 크지만 옷의 색이 서로 섞인다는 단점이 있다. ri, gi, bi를 i 번째 옷의 빨강, 초록, 파랑의 양이라고 하자. n 벌의 옷을 한 번에 세탁하면 색의 섞임 c는

\[c = \sum_{i=1}^{n}{(r_i - r)^2 + (g_i - g)^2 + (b_i - b)^2}\]

로 정의된다. 여기서 r, g, and b는 각각 ri, gi, bi의 산술평균이다. i 번째 옷은 삼차원 RGB 공간에서 점 (ri, gi, bi)로 표현된다. RGB 공간에서 어떤 세 점도 한 직선 위에 있지 않으며, 어떤 네 점도 한 평면 위에 있지 않음이 보장된다.

세탁기가 전기를 많이 쓰기 때문에 n 벌의 옷을 최대 k 개의 그룹으로 나누어 세탁하려고 한다. 전체 색의 섞임은 각 세탁에서 발생한 섞임의 합이다. n 벌의 옷의 색 정보와 k가 주어질 때 섞임을 최소화하는 프로그램을 작성하여라.

입력

첫째 줄에 두 정수 n과 k가 주어진다. (1 ≤ n ≤ 100; 1 ≤ k ≤ 2) 다음 n 개 줄의 i째 줄에는 세 정수 ri, gi, bi가 주어진다. (0 ≤ ri, gi, bi ≤ 1,000)

출력

최소 섞임을 소수점 아래 여섯째 자리까지 반올림하여 출력한다.
