조금 길다란 넥타이

문제

조금 이상한 발명품 주식회사라고 들어보았나? 이 회사는 “조금 이상한“ 발명품을 만드는 것으로 유명하다. 이 문제에서는 줄여서 JOI 주식화사라고 하자. JOI 주식회사는 새로운 제품 “조금 길다란 넥타이”를 개발했다. 1부터 N + 1까지의 번호가 붙은 N + 1가지의 넥타이가 있고, i번 넥타이의 길이는 Ai이다. (1 ≤ i ≤ N + 1)

JOI 주식회사는 시착 행사를 위해 직원들을 모았다. N명의 직원이 행사에 참여하고, j번째 직원은 처음에 길이 Bj짜리 넥타이를 매고 있다.

시착 행사는 다음과 같이 진행된다.

  1. JOI 주식회사의 사장이 행사에서 사용되지 않을 넥타이를 하나 고른다.
  2. 각 직원은 남은 넥타이 중에서 하나씩 고른다. 서로 다른 두 직원이 같은 넥타이를 고르지 않는다.
  3. 마지막으로 각 직원이 원래 착용하고 있던 넥타이를 벗고 고른 넥타이를 맨다.

만약 길이 b짜리 넥타이를 매고 있던 직원이 길이 a짜리 넥타이를 맨다면 max{a − b, 0}만큼의 불편감을 호소한다. 행사의 불편감은 모든 직원의 불편감의 최댓값으로 정의된다.

JOI 주식회사의 회장이 k번 넥타이를 골랐을 때 얻을 수 있는 행사의 최소 불편감을 Ck라 하자.

행사에서 사용되는 넥타이의 길이와 각 직원이 처음에 매고 있는 넥타이의 길이가 주어졌을 때, C1,C2, … ,CN+1의 값을 구하자.

입력

표준 입력에서 아래 데이터를 읽는다. 주어지는 수는 모두 정수이다.

N
A1 … AN+1
B1 … BN

출력

표준 출력에 C1,C2, … ,CN+1을 공백으로 구분하여 출력한다.

제한

  • 1 ≤ N ≤ 200 000.
  • 1 ≤ Ai ≤ 1 000 000 000 (1 ≤ i ≤ N + 1).
  • 1 ≤ Bj ≤ 1 000 000 000 (1 ≤ j ≤ N).

서브태스크

번호 배점 제한
1 1

N ≤ 10.
2 8

N ≤ 2000.
3 91

추가 제약 조건 없음.

다음은 행사의 한 예이다.

  • 사장이 4번 넥타이를 고른다. 이 넥타이는 행사에서 사용되지 않는다.
  • 첫 번째 직원이 1번, 두 번째 직원이 2번, 세 번째 직원이 3번 넥타이를 고른다.
  • 직원들이 고른 넥타이를 맨다.

이 경우 각 직원의 불편감은 2, 0, 3이다. 따라서 행사의 불편감은 3이다. 다른 방식으로 넥타이를 고르면 불편감을 1로 줄일 수 있다. 한 예는,

  • 사장이 4번 넥타이를 고른다. 이 넥타이는 행사에서 사용되지 않는다.
  • 첫 번째 직원이 2번, 두 번째 직원이 3번, 세 번째 직원이 1번 넥타이를 고른다.
  • 직원들이 고른 넥타이를 맨다.

이 경우 각 직원의 불편감은 1, 1, 0이다. 따라서 행사의 불편감은 1이다. 이때 불편감이 최소이므로, C4 = 1이다.
