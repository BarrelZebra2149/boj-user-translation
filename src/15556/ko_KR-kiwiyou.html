정기권

문제

JOI 군은 1번부터 N번까지 N개의 철도역과, 1번부터 M번까지 M개의 철로가 있는 도시에 산다. i번 철로는 Ai번 역과 Bi번 역을 양방향으로 연결하며, 운임은 두 방향 모두 Ci이다. (1 ≤ i ≤ M)

JOI 군은 S번 역 근방에 살며 T번 역 근방의 IOI 고등학교에 다닌다. JOI 군은 두 역을 오갈 수 있는 정기권을 구매하려고 한다. 정기권을 구매할 때는 S번 역에서 T번 역으로 가는 최소 운임의 경로를 골라야 한다. 정기권을 사용하면 고른 경로상의 모든 철로를 어느 방향으로도 추가 운임 없이 지날 수 있다.

JOI 군은 U번 역과 V번 역 근방에 있는 서점에 자주 간다. 그래서 U번 역에서 V번 역으로 가는 추가 운임이 최소화되도록 정기권을 구매하려고 한다.

U번 역에서 V번 역으로 갈 때는 우선 U번 역에서 V번 역으로 가는 경로를 하나 고른다. 추가 운임은 고른 경로에 포함된 각 i번 철로에 대해 아래 방법으로 구한 값의 합이다.

  • i번 철로가 정기권에서 선택된 경로에 포함된다면 0
  • i번 철로가 정기권에서 선택된 경로에 포함되지 않는다면 Ci

JOI 군은 정기권을 위한 적당한 경로를 골랐을 때 U번 역에서 V번 역으로 갈 때 내는 추가 운임의 최솟값을 알고 싶다.

정기권을 위한 경로를 적당히 골랐을 때 U번 역에서 V번 역까지 추가 운임의 최솟값을 구하자.

입력

표준 입력을 통해 아래 입력이 주어진다.

  • 입력의 첫째 줄에 정수 N, M이 주어진다. 이는 JOI 군이 사는 도시가 N개의 역과 M개의 철로로 구성된다는 뜻이다.
  • 둘째 줄에 정수 S, T가 주어진다. 이는 JOI 군이 S번 역과 T번 역을 오가는 정기권을 구매한다는 뜻이다.
  • 셋째 줄에 정수 U, V가 주어진다. 이는 JOI 군이 U번 역에서 V번 역으로 이동할 때 내는 추가 운임을 최소화하려 한다는 뜻이다.
  • 다음 M줄 중 i째 줄에는 세 정수 Ai, Bi, Ci가 주어진다. i번 철로는 Ai번 역과 Bi번 역을 양방향으로 연결하며, 운임은 두 방향 모두 Ci이다.

출력

표준 출력에 정기권의 경로를 적당히 선택해 얻을 수 있는 U번 역으로부터 V번 역까지의 추가 운임의 최솟값을 출력한다.

제한

  • 2 ≤ N ≤ 100 000
  • 1 ≤ M ≤ 200 000
  • 1 ≤ S ≤ N
  • 1 ≤ T ≤ N
  • 1 ≤ U ≤ N
  • 1 ≤ V ≤ N
  • S ≠ T
  • U ≠ V
  • S ≠ U 또는 T ≠ V
  • JOI 군은 철로를 통해 모든 두 역 사이를 이동할 수 있다.
  • 1 ≤ Ai < Bi ≤ N (1 ≤ i ≤ M)
  • 1 ≤ i < j ≤ M인 i에 대해서, Ai ≠ Aj 또는 Bi ≠ Bj이다.
  • 1 ≤ Ci ≤ 1 000 000 000 (1 ≤ i ≤ M)

서브태스크

번호 배점 제한
1 16

S = U.
2 15

S번 역에서 T번 역으로 가는 최소 운임 경로는 유일하다.
3 24

N ≤ 300
4 45

추가 제약 조건 없음

JOI 군이 정기권 구매 시에 고를 수 있는 경로는 1번 역 → 2번 역 → 3번 역 → 5번 역 → 6번 역으로 유일하다.

1번 역부터 4번 역까지의 추가 운임을 최소화하려면 1번 역 → 2번 역 → 3번 역 → 5번 역 → 4번 역과 같이 이동하면 된다. 이때 추가 운임은

  • 4번 역과 5번 역을 연결하는 5번 철로의 경우 2
  • 다른 철로의 경우 0

으로 총 2이다.

3번 역에서 6번 역으로 이동할 때 정기권을 사용할 수 없다.
