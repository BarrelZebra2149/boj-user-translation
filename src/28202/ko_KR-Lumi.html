Classical A+B Problem

문제

정수 중에 repdigit이라 불리는 특별한 정수는 다음과 같은 조건을 만족한다.

    양의 정수이다.
    10진법으로 나타내었을 때, 똑같은 숫자가 반복되는 형태이다.

예를 들어, 1, 666, 4444, 999999는 repdigit이고, 0, 44244, 50216, 787788은 repdigit이 아니다.

양의 정수 n이 주어질 때, n = a + b를 만족하는 두 repdigit a, b를 찾아라. n은 항상 두 repdigit a, b의 합으로 나타낼 수 있음이 보장되며, 가능한 조합이 여러 개일 때는 그 중에 하나만 찾으면 된다.

입력

각각의 테스트는 여러 개의 테스트 케이스로 구성되어있다.

    첫번째 줄에는 테스트 케이스의 수 t가 주어진다.
    그 이후로 t줄에 걸쳐 각각의 테스트 케이스가 한 줄에 하나의 정수 n이 각각 주어진다. (2 ≤ n < 104000)

n은 항상 두 repdigit a, b의 합으로 나타낼 수 있음이 보장된다. 또한, 모든 테스트 케이스에 걸쳐 n의 자릿수의 총합은 105을 넘지 않음이 보장된다.

출력

매 테스트케이스마다 n = a + b를 만족하는 두 repdigit a, b를 출력하라. 가능한 조합이 여러개라면 개중에 하나만 출력하라.
