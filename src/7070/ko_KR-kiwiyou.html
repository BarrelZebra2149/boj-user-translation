카드 섞기 게임

문제

수용이와 명현이는 1부터 N까지의 수가 쓰인 카드 N장과 카드 섞는 기계를 가지고 있다. N은 홀수라고 가정하자.

기계는 카드 묶음을 받아 더블 셔플을 수행한다. 모든 위치 i (1 ≤ i ≤ N)에 대해서, 만약 위치 i에 있는 카드의 수가 j이고 위치 j에 있는 카드의 수가 k라면, 더블 셔플 이후 위치 i에는 k가 쓰인 카드가 놓인다.

수용이와 명현이는 게임을 하려고 한다. 처음에 수용이가 1부터 N까지의 수를 원하는 순서로 써서 수열 a1, a2, …, aN을 만든다. 다음으로 위치 모든 i (1 ≤ i ≤ N-1)에 대해 위치 ai에 수 ai+1이 적힌 카드를 놓는다. 위치 aN에는 수 a1이 쓰인 카드가 놓인다.

위치 i에 놓인 카드의 수를 xi라고 하면, 카드 묶음은 수열 x1, x2, …, xN으로 나타낼 수 있다.

수용이는 기계를 통해 더블 셔플을 S번 수행한다. 그러면 카드 묶음을 나타내는 수열이 p1, p2, …, pN 이 되고, 수용이는 이 수열과 더블 셔플의 횟수 S를 명현이에게 알려준다. 명현이는 섞기 전 카드 묶음을 나타내는 수열 x1, x2, …, xN을 맞혀야 한다.

 

입력

첫째 줄에 카드의 수를 나타내는 홀수 N (1 ≤ N ≤ 1000)과 더블 셔플의 횟수 S (1 ≤ S ≤ 1000)이 주어진다.

다음 N개의 줄에는 더블 셔플을 진행한 후 카드 묶음을 나타내는 수열 p가 주어진다. i+1번째 줄에는 위치 i에 놓인 카드에 적힌 수 pi가 주어진다. (1 ≤ i ≤ N)

출력

카드를 섞기 전 카드 묶음을 나타내는 수열을 N줄에 걸쳐 출력한다.

i번째 줄에는 위치 i에 놓인 카드에 적힌 수 xi를 출력해야 한다. (1 ≤ i ≤ N)
