이름 생성기

문제

Hank Morebooks는 유명한 작가다. 그는 새 책을 썼는데, 등장인물 k명의 이름을 아직 정하지 못했다. 그는 등장인물의 이름을 다음과 같은 방법으로 정하기로 했다.

먼저 영어 소문자로 이루어진 문자열을 하나 정하고, 이를 k개의 문자열로 나눈다. 이때, 빈 문자열이 있어서는 안 된다. 그가 주어진 문자열을 k개의 서로 다른 이름으로 나눌 수 있는지 알려주자.

입력

첫 번째 줄에 테스트 케이스의 수 t (1 ≤ t ≤ 1000)가 주어진다.

각 테스트 케이스는 두 줄로 주어진다. 첫 번째 줄에 나눌 문자열이 주어진다. 이 문자열의 길이는 최대 100이고, 영어 소문자로만 이루어져 있다. 두 번째 줄에 정수 k (1 ≤ k ≤ 5)가 주어진다.

출력

각 테스트 케이스마다 다음과 같이 출력한다.

주어진 문자열을 k개의 서로 다른 이름으로 나눌 수 있다면 첫 번째 줄에 "YES"를 출력한다. 그리고 다음 k개의 줄에 나눈 k개의 문자열을 순서대로 출력한다.

주어진 문자열을 k개의 서로 다른 이름으로 나눌 수 없다면 첫 번째 줄에 "NO"를 출력한다.
